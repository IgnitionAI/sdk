# Contributing

The public SDK accepts fixes, documentation improvements and client-side
features that use the signed public contract. Server implementation, licensing,
administration, MCP and widget internals remain in the private product.

Before opening a pull request:

1. Install with `bun install --frozen-lockfile`.
2. Run `bun run verify:contract`.
3. Run `bun run typecheck`, `bun test` and `bun run build`.
4. Run `bun audit` and `bun pm pack --dry-run`.

Files under `src/generated/` are generated by `bun run generate:types` and must
not be edited manually. Contract changes arrive through a signed synchronization
pull request from the private IgnitionRAG product repository.

By contributing, you agree that your contribution is licensed under MIT.
